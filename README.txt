TEST TECHNIQUE JAVA ADNEOM BERNET QUENTIN

La documentation du package com.adneom.test.quentin se trouve dans le dossier doc.

Les sources se trouvent dans le dossier com (projet Eclipse).
 
La classe contenant l’algorithme de la solution (et la fonction « partition ») est la classe : com.adneom.test.quentin.Solver

La classe com.adneom.test.quentin.Main execute la solution sur l’exemple donné dans l’énoncé. C’est le point d’entré du projet. 

La classe com.adneom.test.quentin.Tests contient 8 tests unitaires écrit avec JUNIT4 et test la fonction « partition ».
